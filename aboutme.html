Tools used

Developed using JavaScript, css and html. Used the Leaflet API for map data and functionality. Incorperated Location IQ for location retrevial and centering of map on user search. AQ Platform API was used to locate air quality measurements from various stations based on map location (Center and radius of map). Vue was used for more simple data aquisitioning and rendering throughout the program. The Leaflet head API was used for creating a heatmap based off measurements that were taken from the air quality API.
